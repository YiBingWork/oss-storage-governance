# OSS Storage Governance Platform

阿里云 OSS 存储治理 Web 化平台 — 大数据资产治理平台

## 技术栈

- **后端：** Java Spring Boot 3.x + MyBatis-Plus
- **前端：** Vue 3 + Element Plus + ECharts
- **数据库：** MySQL
- **部署：** Docker Compose

## 功能模块

1. 全局看板 (Global Dashboard)
2. 成本中心 (Cost Center)
3. 表级分析 (Table-Level Insight)
4. 小文件治理 (Small File Center)
5. 生命周期建议 (Lifecycle Advisor)
6. 无用数据识别 (Junk Data Detection)
